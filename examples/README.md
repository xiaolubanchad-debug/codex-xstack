# Node-first examples plan / Node-first 示例计划

This directory contains example usage patterns for xstack.  
这个目录用于存放 xstack 的示例使用方式。

## Current examples / 当前已有示例

- `frontend-feature.md`
- `backend-api-change.md`
- `fullstack-flow.md`

These are still useful, but they are intentionally generic.  
这些示例仍然有用，但它们目前仍偏通用。

## Next step / 下一步

For the Node-first direction, examples should become more stack-specific.  
随着项目转向 Node-first，示例也应该更偏向具体技术栈。

## What each example should include / 每个示例应该包含什么

Each future example should include:  
后续每个示例都应该包含：

- the scenario / 场景说明
- the recommended skill flow / 推荐的 skill 流程
- the relevant packs / 对应使用的 packs
- example prompts / 示例提示词
- what xstack should catch / xstack 应该帮你抓到什么
- what a good final result looks like / 理想结果长什么样

## Goal / 目标

The goal is to make xstack easier to adopt for real Node teams by showing realistic stack-shaped usage instead of only abstract workflow guidance.  
目标是让真实的 Node 团队更容易采用 xstack：不只是看到抽象流程说明，而是看到贴近真实技术栈的使用方式。
