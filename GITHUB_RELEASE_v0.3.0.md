# xstack v0.3.0 — Node-first workflow stack

## English

xstack v0.3.0 is the release where xstack becomes clearly **Node-first**.

What began as a compact web workflow layer has now grown into a more concrete **Node/TypeScript delivery stack for Codex**, with stack-specific packs, stronger onboarding, and real workflow examples.

### Highlights

#### Node-first stack packs

- `nextjs-app-pack`
- `prisma-pack`
- `nestjs-backend-pack`
- `trpc-pack`
- `express-api-pack`

#### Web delivery packs

- `frontend-state-flow-pack`
- `form-flow-pack`
- `component-consistency-pack`
- `backend-api-pack`
- `auth-permission-pack`
- `db-migration-pack`
- `fullstack-contract-pack`
- `release-risk-pack`
- `incident-response-pack`

#### Review skills

- `review-frontend-flow`
- `review-backend-api`

#### Onboarding and workflow improvements

- bilingual README
- bilingual release notes
- bilingual roadmap and contributing docs
- install script
- doctor script
- quick start guidance
- Node-first examples hub

#### Real Node-first examples

- `nextjs-prisma-feature-flow.md`
- `nextjs-trpc-prisma-fullstack-flow.md`
- `nest-prisma-backend-flow.md`
- `express-prisma-api-flow.md`

### Why this release matters

v0.3.0 is not just about adding more packs.

It is the release where xstack starts to feel like a real operating layer for teams building Node web apps with Codex:

- clearer positioning
- stronger stack support
- better install and onboarding flow
- more realistic workflow examples
- better guidance around review, QA, and release judgment

### What’s next

Likely next directions after v0.3.0:

- deeper Node-specific examples
- continued improvement of onboarding and validation flows
- selective expansion of Node-first packs only where they improve real workflow quality

### 为什么这个版本重要

v0.3.0 的意义不只是“加了更多 packs”。

它真正代表的是：xstack 开始更像一层面向真实 Node Web 团队的操作层，帮助他们用 Codex 做更有结构的开发：

- 定位更清晰
- 技术栈支持更具体
- 安装与上手路径更完整
- workflow 示例更贴近真实开发
- review、QA 和发版判断更有抓手

### 下一步

v0.3.0 之后比较自然的方向包括：

- 更深入的 Node 专项示例

- 只在能真实提升工作流质量时，继续扩展 Node-first packs
