# xstack v0.3.0

## Summary / 概述

xstack v0.3.0 is the release where xstack becomes clearly **Node-first**.  
xstack v0.3.0 是 xstack 明确转向 **Node-first** 的版本。

What began as a compact web workflow layer has now grown into a more concrete **Node/TypeScript delivery stack for Codex**.  
它已经不再只是一个紧凑的 Web workflow 层，而开始成长为一个更具体的、面向 Codex 的 **Node/TypeScript 交付栈**。

## Why this release matters / 这个版本为什么重要

v0.3.0 is not just a larger pack set. It is the release where xstack starts to feel like a practical operating layer for teams building real Node web apps with Codex.  
v0.3.0 的意义不只是“pack 更多了”。它真正代表的是：xstack 开始更像一层面向真实 Node Web 团队的操作层，帮助他们用 Codex 做更有结构的开发。

This release improves:  
这个版本重点提升的方面包括：

- Node-first positioning / Node-first 定位
- stack-specific workflow support / 技术栈专项工作流支持
- onboarding quality / 上手体验质量
- workflow examples / 工作流示例体系
- review, QA, and release judgment / review、QA 和发版判断能力

## Notes / 说明

v0.3.0 is still intentionally compact.  
v0.3.0 仍然刻意保持紧凑。

The goal is not to support every ecosystem. The goal is to make xstack meaningfully stronger for real Node/TypeScript teams using Codex.  
目标不是支持所有生态，而是让 xstack 对真实使用 Codex 的 Node/TypeScript 团队更有价值。
